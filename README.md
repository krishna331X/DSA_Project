# 💅 Glow & Gloss - C++ Salon Engine

This is a high-performance **C++ Backend System** for a luxury Nail Studio. It demonstrates the use of core Data Structures to handle real-world business logic.

## 🚀 Key Features
- **Scheduling (Linked List):** Handles dynamic appointment bookings without memory waste.
- **Waitlist (Queue):** Implements FIFO (First-In-First-Out) for walk-in customers.
- **Activity Logs (Stack):** Keeps track of recent actions for Undo/Audit purposes.
- **Search Engine:** Linear search implementation to find customer records.

## 🛠 Technical Details
- **Language:** C++17
- **DSA Used:** Singly Linked List, STL Queue, STL Stack, Search Algorithms.
